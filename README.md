# Adventure_Works_DataEngineering_Project

An end-to-end data engineering solution built on Azure, showcasing data ingestion, transformation, storage, and visualization. This project leverages Azure Data Factory for pipeline orchestration, Azure Data Lake Gen2 for scalable storage, Azure Data-bricks for data transformation, and Azure Synapse Analytics for advanced analytics. 

# Tech Stack
Microsoft Azure
Azure Data Factory
Azure Databricks
Azure Synapse Analytics
Power BI
